===========================
Getting Started with Django
===========================

In this section you'll learn about Django's core components and how they work together to create a web application. By the end of this section you'll have built and deployed a simple application and will understand:

* how Django works with databases
* how to respond to HTTP requests
* how to process user input
* how Django integrates with CSS and other assets
* how to deploy to Heroku and XXX
